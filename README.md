# CLI Client App on Ubuntu for cloud189 cloud disk

## Functions

### Login

Support login to cloud disk natively on Ubuntu

### List

List the dirs and files on cloud disk

### Download

Download the dir or file from cloud disk

### Upload

Upload the dir or file to cloud disk

### Incremental Upload

Bind local dir to remote dir, once files in dir is updated, only upload the modified files to cloud disk

### Incremental Download

Similar to Incremental Upload, but download

### Status

View the upload or download status
